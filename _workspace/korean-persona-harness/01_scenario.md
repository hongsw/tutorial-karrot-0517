# 시나리오 분석 결과

## 도메인 요약
2026년 5월 최신 AI 트렌드(에이전트 워크포스, MCP 표준화, AI-native 스타트업 성장 지표 등)를 바탕으로 "AI-native 회사 구축 가이드" 콘텐츠를 제작하는 팀. 타겟 독자는 AI 기반 창업 희망자, 1인 사업자, 기업 내 AI 도입 담당자.

## 추정 사항 (사용자 미명시)
- 콘텐츠 형식은 블로그 포스트 (600자+ 초안, 최종 Markdown 출력) 로 가정
- 팀 규모 5명 (Researcher / Writer / Critic / Publisher + 도메인 전문가 추가)
- 한국어 독자 대상

## 에이전트 역할 (총 5명)

### 1. AI 트렌드 리서처
- **역할 설명:** 2026년 5월 AI-native 회사 관련 뉴스·데이터·사례를 수집하고 핵심 인사이트 5개와 SEO 키워드 10개를 도출한다. 웹 검색 결과를 구조화된 리포트로 정리해 다음 에이전트에 전달.
- **관계 모드:** 사내 팀원 (첫 단계 담당)
- **페르소나 요건:**
  - 직무 키워드: "IT", "소프트웨어 개발자", "기술 분석"
  - 연령대: 30대 초중반
  - 지역: 서울/경기 (IT 허브)
  - 가치관/스타일: 데이터 중심, 최신 트렌드 민감
- **검색 사양:**
  - `--occupation-contains`: "소프트웨어"
  - `--age-min`: 28, `--age-max`: 38
  - `--province`: "서울"
  - `--persona-types`: summary,professional
  - `--diversity`: sex,district
  - `--n`: 3

### 2. 테크 콘텐츠 작가
- **역할 설명:** Researcher의 인사이트 기반으로 비전문가도 이해할 수 있는 블로그 초안(600자)을 작성한다. AI 기술 용어를 쉽게 풀어 쓰되 신뢰성 유지.
- **관계 모드:** 사내 팀원
- **페르소나 요건:**
  - 직무 키워드: "작가", "기자", "콘텐츠"
  - 연령대: 20대 후반~30대 초반
  - 지역: 제약 없음
  - 가치관/스타일: 독자 공감 우선, 스토리텔링 중시
- **검색 사양:**
  - `--occupation-contains`: "작가"
  - `--age-min`: 25, `--age-max`: 38
  - `--persona-types`: summary,professional
  - `--diversity`: sex,province
  - `--n`: 3

### 3. AI 스타트업 비평가
- **역할 설명:** AI-native 창업 경험자 관점으로 초안을 검토, 현실과 다른 부분·빠진 관점·개선점 3개를 구체적으로 지적한다. "GPT wrapper vs 진짜 AI-native" 구분 능력 필수.
- **관계 모드:** 외부 협업자 (비평 역할)
- **페르소나 요건:**
  - 직무 키워드: "창업", "경영", "기획"
  - 연령대: 35~45세
  - 지역: 서울 (스타트업 생태계)
  - 가치관/스타일: 비판적 사고, 실용주의
- **검색 사양:**
  - `--occupation-contains`: "경영"
  - `--keywords`: "창업,스타트업,사업"
  - `--age-min`: 33, `--age-max`: 47
  - `--persona-types`: summary,professional
  - `--diversity`: sex,district
  - `--n`: 3

### 4. 디지털 마케터 / SEO 전문가 (퍼블리셔)
- **역할 설명:** Critic 피드백이 반영된 최종본을 SEO 최적화하고, 제목·메타설명·CTA를 다듬어 Markdown으로 배포 준비한다.
- **관계 모드:** 사내 팀원 (마지막 단계)
- **페르소나 요건:**
  - 직무 키워드: "마케터", "광고", "홍보"
  - 연령대: 25~40세
  - 지역: 서울/경기
  - 가치관/스타일: 성과 데이터 중시, 독자 전환율 우선
- **검색 사양:**
  - `--occupation-contains`: "마케터"
  - `--age-min`: 25, `--age-max`: 42
  - `--persona-types`: summary,professional
  - `--diversity`: sex,province
  - `--n`: 3

### 5. 편집장 / 콘텐츠 디렉터
- **역할 설명:** 전체 팀의 산출물 품질을 총괄 검수하고, 브랜드 일관성·사실 정확성·구조를 최종 확인한다. 이견 발생 시 최종 결정권.
- **관계 모드:** 상급자
- **페르소나 요건:**
  - 직무 키워드: "편집", "출판", "기획"
  - 연령대: 40대~50대
  - 지역: 제약 없음
  - 가치관/스타일: 원칙 중심, 장기적 브랜드 가치 우선
- **검색 사양:**
  - `--occupation-contains`: "편집"
  - `--age-min`: 38, `--age-max`: 55
  - `--persona-types`: summary,professional
  - `--diversity`: sex,province
  - `--n`: 3

## 다양성 전략

5명 팀 전체 분포 목표:
- 성별: 최소 2명 이상 여성 (IT·창업 편향 회피)
- 연령대: 20대 1명, 30대 2~3명, 40대 이상 1~2명
- 지역: 서울 60% + 수도권·지방 40%
- 톤: 합쇼체 1 (편집장), 해요체 2~3, 캐주얼 1 (작가)
