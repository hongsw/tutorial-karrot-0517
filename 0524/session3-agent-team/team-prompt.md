## 프롬프트 1: 콘텐츠 팀 구성 + 실행

```
당신은 4명 전문가 에이전트 팀의 Conductor입니다.
각 에이전트가 순서대로 작업하고, 결과를 다음 에이전트에게 넘깁니다.

[Researcher] 주제를 리서치해 핵심 포인트 5개와 SEO 키워드 10개를 뽑습니다.
[Writer]     Researcher 결과로 블로그 초안(600자)을 작성합니다.
[Critic]     Writer 초안의 개선점 3개를 구체적으로 제시합니다.
[Publisher]  Critic 피드백 반영해 최종본을 완성하고 Markdown으로 출력합니다.

주제: "1인 사업자가 AI로 고객 응대를 자동화하는 3가지 방법"

각 에이전트 앞에 [에이전트명] 태그를 붙여 순서대로 진행해주세요.
```

## 프롬프트 2: 고객 CS 팀

```
당신은 고객 CS 에이전트 팀 Conductor입니다.

[Triager]   이메일을 읽고 유형 분류 (문의/불만/계약/스팸) + 우선순위 판단
[Responder] 분류 결과로 답장 초안 작성 (한국어, 공손하게)
[Escalator] 불만/계약 건은 담당자 에스컬레이션 메모 작성

입력 이메일:
"안녕하세요. 지난주에 신청한 컨설팅 일정이 아직 확정이 안 됐는데,
언제쯤 연락 받을 수 있을까요? 이번 주 안에 진행이 필요합니다."

각 에이전트 결과를 순서대로 보여주세요.
```
