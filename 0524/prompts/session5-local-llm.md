## 프롬프트 A: 내 업무에 로컬 LLM이 필요한지 진단

```
로컬 LLM(내 컴퓨터에서 실행하는 AI)과 클라우드 AI(ChatGPT, Claude)의 차이:

로컬 LLM 장점:
- 토큰 비용 없음 (무제한 사용)
- 인터넷 없이 작동
- 개인정보/기밀 데이터 외부 전송 없음
- Ollama, LM Studio로 쉽게 설치 가능

로컬 LLM 단점:
- 성능이 GPT-4/Claude보다 낮음
- 좋은 컴퓨터 필요 (RAM 16GB+ 권장)
- 한국어 품질이 클라우드보다 약함

baryon.ai TalkMode:
- 한국어 특화 Qwen3-Coder 기반
- 브라우저에서 무료 사용 (uuidgen으로 Device ID 등록)
- API 호환 (opencode, continue 등 IDE 툴 연동 가능)

[내 정보]
- 주요 작업: (예: 계약서 검토, 고객 데이터 분석, 코드 작성, 일반 글쓰기)
- 다루는 데이터 민감도: (예: 고객 개인정보 있음 / 공개 가능한 내용만 / 기밀 계약 포함)
- 컴퓨터 사양: (예: MacBook M2 16GB / Windows 16GB RAM / 구형 PC 8GB)
- 현재 AI 월 지출: (예: ChatGPT Plus $20, Claude Pro $20)

내 상황에서 로컬 LLM이 도움이 될지 판단해주세요.
어떤 작업은 로컬로, 어떤 작업은 클라우드로 나눠야 하는지 기준도요.
```

## 프롬프트 B: 개인정보 포함 데이터 처리 (로컬 LLM 시나리오)

```
아래 고객 데이터를 분석해야 합니다. 외부 서버로 보내면 안 됩니다.
(실제로는 이걸 로컬 LLM이나 baryon.ai에서 처리합니다.)

[가상 데이터 — 실습용]
고객명: 홍길동, 연락처: 010-1234-5678, 구매이력: 강의A 3회, 강의B 1회
고객명: 김영희, 연락처: 010-9876-5432, 구매이력: 강의A 1회, 강의C 2회
고객명: 이철수, 연락처: 010-5555-1234, 구매이력: 강의B 2회, 강의C 1회

이 데이터로:
1. 고객별 관심사 분류 (강의 패턴 기반)
2. 재구매 가능성 높은 고객 순위
3. 각 고객에게 보낼 맞춤 이메일 초안 1줄

만들어주세요.

(실습 포인트: 이런 데이터를 ChatGPT에 올리면 안 됩니다.
 로컬 Ollama나 baryon.ai는 외부 서버 없이 처리 가능합니다.)
```

## 프롬프트 C: 도구별 선택 가이드

```
다음 3가지 로컬/무료 LLM 옵션을 비교해주세요:

1. Ollama (로컬 설치)
   - 모델: Qwen3:14b (코딩), Llama3.2:3b (가벼움), Gemma3:27b (한국어)
   - 설치: ollama.com에서 다운로드 후 ollama run 모델명

2. LM Studio (GUI 있는 로컬 LLM)
   - 모델 허브에서 선택해서 다운로드
   - OpenAI 호환 API 제공

3. baryon.ai TalkMode (클라우드지만 무료)
   - Qwen3-Coder 기반, 한국어 양호
   - Device ID만 등록하면 무료 사용

[내 정보]
- 주 사용 목적: (예: 코드 작성 보조 / 문서 요약 / 한국어 글쓰기)
- 컴퓨터: (예: Mac M3 Pro 36GB / Windows RTX 4080)
- 기술 수준: (예: 터미널 써본 적 없음 / 기본 명령어 가능 / 개발자)

내 상황에 가장 맞는 옵션과 설치 시작 방법을 알려주세요.
```
