## 프롬프트 A: 구축 방법 진단 + 로드맵

```
AI 에이전트 팀을 구축하는 4가지 방법이 있습니다:

A. Vibecoding (Claude Code / Cursor)
   - 코딩 없이 대화로 앱 생성
   - 비용: Claude Max $100/월 또는 Cursor Pro $20/월
   - 적합: 맞춤형 도구 필요, 빠른 프로토타입

B. Claude API 직접 사용
   - Python/JS로 직접 연동, 가장 유연
   - 비용: 사용량 기반 (claude-sonnet-4-6 기준 입력 $3/1M 토큰)
   - 적합: 기술자, 반복 자동화, 비용 최적화

C. CrewAI (Python 에이전트 프레임워크)
   - Python 코드로 에이전트 팀 구성
   - 비용: 오픈소스 무료 + API 비용
   - 적합: Python 할 줄 아는 사람, 복잡한 다중 에이전트

D. n8n (노코드 시각적 자동화)
   - 블록 연결로 워크플로우 구성
   - 비용: 셀프호스팅 무료 / 클라우드 $20/월~
   - 적합: 비개발자, 기존 도구 연동(이메일, 시트, Slack)

[내 정보]
- 직업/사업: (예: 세무사 1인 사무소)
- 만들고 싶은 것: (예: 고객 문의 자동 분류 + 답변 초안 생성)
- 기술 수준: (예: 코딩 전혀 못함 / 파이썬 기초 / 실무 개발자)
- 월 예산: (예: 5만원 이하 / 20만원 / 상관없음)

내 상황에 맞는 구축 방법을 추천해주세요.
왜 그 방법이 맞는지, 첫 2주 실행 계획도 알려주세요.
```
